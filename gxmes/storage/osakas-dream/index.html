Osaka's Dream (Full Game)

Osaka's Dream is a fan project made after numerous requests received on  a reel uploaded to Instagram showing footage of the character, Osaka, flying around in a game that didn't yet exist.

Now, this game exists as a full game featuring 7 levels containing 5 stages and 1-2 bosses each, as well as a lot of story content in the form of a visual novel.  Although this game is finished, you can assume more content is on it's way.

The main objective of the game is to go through each stage, while collecting Osaka's favorite food 'Sata Andagi'. The main story goes as follow:

"WAHHH!! MAH' FRIENDS HAVE BEEN KIDNAPPED BY A BUNCH OF ANIMALS AND NOW AH' GOTTA SAVE 'EM!!!"

Default Controls (Controls can be changed in the settings):

WASD: Fly around

Shift: Dash in the direction you're moving.

/: Use currently equipped powerup

P: Pause the game. Use M to return to the main menu.
